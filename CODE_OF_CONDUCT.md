# Code of Conduct

## Our Commitment

ArchitectOS is committed to creating an open, welcoming, and respectful community for everyone interested in software architecture, engineering tools, and open-source development.

We value collaboration, diversity of ideas, and constructive technical discussion.

---

# Expected Behavior

Community members are expected to:

- Be respectful and professional.
- Provide constructive feedback.
- Focus on technical discussions.
- Welcome different perspectives.
- Help others learn and contribute.

---

# Unacceptable Behavior

The following behaviors are not acceptable:

- Harassment or personal attacks.
- Discriminatory language.
- Trolling or intentionally disruptive behavior.
- Sharing inappropriate content.
- Disrespectful communication.

---

# Technical Discussions

ArchitectOS encourages healthy technical debate.

Disagreements about:

- Architecture choices
- Design decisions
- Implementation approaches

should be handled respectfully and based on technical reasoning.

---

# Enforcement

Project maintainers may take appropriate action when community guidelines are violated.

The goal is to maintain a productive and welcoming environment for all contributors.

---

# Scope

This code of conduct applies to:

- GitHub issues
- Pull requests
- Discussions
- Community interactions related to ArchitectOS
